# Automation Habi

En este proyecto podras encontrar una automatización del aplicativo ToolsQA, realizada para la compañia de HABI

## Tecnologías

Lenguaje: Java
Framework: Serenity
Orquestador de dependencias: Maven
Runner: JUnit
Pattern Design: ScreePlay
